# BetterPlayerHUD v1.6.0 使用说明书

> Minecraft 1.8.9 Forge 客户端模组
> 作者：黄锦造办，吕芳审阅，嘉靖朝司礼监出品

## 一、安装

1. 确保已安装 **Minecraft Forge 1.8.9**（推荐 1.8.9-11.15.1.2318 及以上版本）
2. 将 `BetterPlayerHUD-1.6.0.jar` 放入 `.minecraft/mods/` 目录
3. 启动游戏，模组自动加载

---

## 二、快速上手

### 快捷键一览

| 按键 | 功能 | 说明 |
|------|------|------|
| **F8** | 打开配置 GUI | 默认未绑定，需在 MC 控制设置中手动设键 |
| **F9** | 切换中/英文 | 仅在配置 GUI 打开时生效 |
| **F7** | HUD 编辑模式 | 拖拽各模块位置，Ctrl+滚轮调大小 |
| **Esc** | 关闭配置 GUI | — |

> 注：F8 默认 `KEY_NONE`，请到 MC 主菜单 → 设置 → 控制 →「Better Player HUD」→「打开配置界面」中绑定你想要的热键（推荐 F8）。

### 首次使用流程

1. 按 **F8**（或你绑定的键）打开配置界面
2. 浏览各 Tab 开关，按需开启/关闭模块
3. 点「保存并关闭」生效
4. 按 **F7** 进入编辑模式，拖拽各 HUD 元素到喜欢的位置

### 五个 Tab

| Tab | 内容 |
|-----|------|
| **通用** | 罗盘、状态栏（血/饱食度）、距离信息、按键显示、全局开关 |
| **聊天** | ChromaChat 全部配置：宽度/行数/时间戳/头像/告示牌合并/动画 |
| **战斗** | 准星样式、命中标识、连击、方块描边、实体高亮、RGB 流光、方块破坏指示器 |
| **性能** | 药水效果、装备栏、物品数量、危机警戒、经验条、Boss血条、快捷栏、Tab列表、调试信息、**新增：跳跃条/氧气条/坐骑血量** |
| **颜色** | 各模块颜色统一配置（状态栏/罗盘/药水/准星/命中标识/方块破坏等） |

### 搜索功能

在配置 GUI 顶部的搜索框中输入文字，实时过滤配置条目，按名称和 tooltip 匹配。按 Enter 或 Esc 退出搜索模式。

### 数值调整方式

| 操作 | 步进 |
|------|------|
| 点击 `[<]` / `[>]` 箭头 | 基础步进 |
| Shift + 箭头 | ×10 |
| Ctrl + 箭头 | ×50（或 ×0.01 浮点） |
| Shift + Ctrl + 箭头 | ×100 |

## 四、全部 18 个 HUD 模块

### 4.1 准星 (Crosshair)
- 7 种样式：点 / 十字 / 十字间隙 / CSGO / 圆 / 菱形 / 三角
- **动态扩散**：行走/跳跃/拉弓时准星根据速度自动扩散
- RGB 流光，旋转角度，描边开关

### 4.2 血量 (Health)
- 自定义血条 + 数字 + 头像
- **三种血条样式**：
  - **Default**：经典实心条
  - **Modern**：渐变 + 顶部高光
  - **Pixel**：像素块状交替色
- 低血量闪烁警告

### 4.3 饱食度 (Food)
- 橙色饥饿度条，可开关

### 4.4 护甲 (Armor)
- 护甲值显示 + 耐久条

### 4.5 药水效果 (Potions)
- 药水图标 + 计时器显示剩余时间
- **药水预警**：剩余时间低于阈值时图标闪烁红色警告

### 4.6 装备栏 (Equip)
- 四件装备图标显示
- **盔甲耐久**：装备图标下方显示剩余耐久数字（绿/黄/红三色）

### 4.7 经验条 (ExpBar)
- 深蓝渐变经验条 + 等级数字 + 满级闪烁

### 4.8 Boss 血条 (BossHP)
- 红色 Boss 血量条

### 4.9 快捷栏 (Hotbar)
- 9 槽自定义快捷栏，带物品图标和耐久条

### 4.10 玩家列表 (TabList)
- 按队伍分组的玩家列表，Tab 键按住时显示

### 4.11 调试信息 (Debug)
- FPS / 坐标 / 内存 / 朝向

### 4.12 跳跃条 (JumpBar)
- 跳跃蓄力条：按住空格时 0→1 循环蓄力
- 颜色从绿到红渐变

### 4.13 氧气条 (Air)
- 水下时显示自定义氧气条
- 蓝→红渐变 + 百分比

### 4.14 坐骑血量 (MountHP)
- 骑乘时显示坐骑血量条
- 名称 + 血量百分比 + 三色阈值

### 4.15 连击计数 (Combo)
- 命中连击计数显示

### 4.16 距离信息 (Distance)
- 看向的目标距离 + 方块坐标/名称

### 4.17 按键显示 (Keys)
- WASD 等按键操作实时显示

### 4.18 危机警戒 (Crisis)
- 低血量 / 附近 TNT 等危险情况屏幕警告

## 五、ChromaChat 聊天系统

### 概述

完全替代原版聊天框的现代聊天系统，支持：

- **时间戳**：三种格式可选 `[MM-dd HH:mm]` / `[HH:mm]` / `[HH:mm:ss]`
- **头像**：消息左侧显示发送者皮肤头像
- **告示牌合并**：自动合并 Hypixel 等服务器的系统公告板（装饰线、居中文本、XP消息等），整块灰色显示
- **消息折叠**：相同发送者同内容自动折叠，显示计数重叠角标
- **弹入动画**：新消息弹性入场动画，时长可调
- **悬停高亮**：鼠标悬停消息高亮
- **颜色自定义**：所有颜色均可通过 ARGB 滑块取色器调整

### 聊天顺序

最新消息在最下方，最旧消息在最上方（标准 Minecraft 聊天顺序）。支持向上滚动查看历史。

## 六、HUD 编辑模式 (F7)

按 **F7** 进入编辑模式，各 HUD 模块周围显示虚线框：

- **拖拽**：按住模块边缘拖拽移动位置
- **Ctrl + 滚轮**：调整模块大小（支持缩放的模块）
- **右键点击**：复位到默认位置
- **再次按 F7**：退出编辑模式

## 七、常见问题

### Q: 为什么按 F8 没反应？

F8 默认未绑定。第一次使用需到 MC 控制设置中绑定热键：
`设置 → 控制 → Better Player HUD → 打开配置界面 → 设置按键`

### Q: 怎么切换语言？

在配置 GUI 中点击右上角的「中/EN」按钮，或按 **F9** 快捷键。

### Q: 改了配置后怎么生效？

点「保存并关闭」按钮。如果不小心点了「取消」，所有修改丢弃。

### Q: 血条样式在哪里切换？

配置 GUI → 通用 Tab → 找到「血条样式」条目，点 `[<]` / `[>]` 在 default / modern / pixel 间切换。

### Q: ChromaChat 聊天框太宽了？

配置 GUI → 聊天 Tab → 调整「聊天框宽度」滑块（80~800 像素）。

### Q: 配置全乱了怎么办？

配置 GUI → 点「重置全部」按钮，所有配置恢复出厂设置。
