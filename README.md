# BetterPlayerHUD

一个轻量化的 Minecraft 1.8.9 用户界面模组。

## 缘起

鉴于 Essential 模组逐渐走到了轻量化的反面，各种各样的强制捆绑层出不穷，遂制作了 BetterPlayerHUD——一半为了练习，一半为了自用。

## 功能一览

| 模块 | 说明 |
|------|------|
| 🧭 罗盘 | 黑色行动冷战式指针罗盘，支持方向标记、刻度、精确角度 |
| ❤️ 玩家信息 | 血量 / 护甲 / 饱食度（含 Saturation 叠层）/ 头像，百进制显示 |
| 📏 距离 & 方块信息 | 目标距离、方块坐标、硬度、所需工具 |
| ⌨️ 按键显示 | 实时按键操作显示 + CPS 计数（滑动窗口） |
| 🎯 方块/实体描边 | 方块描边 + 实体碰撞箱（仅在瞄准时触发，隐身碰撞箱隐藏可选）|
| 🌈 RGB 流光 | 描边/碰撞箱动态逐点 HSV 着色，64段细分，正弦波/线性流动 |
| 🎪 目标血量 | 自定义目标血量条，支持玩家/生物/Boss |
| 📊 性能检测 | FPS / 坐标 XYZ / TPS / Ping / 服务器 IP，每项独立开关 |
| 🎯 自定义准星 | 7 种样式（点/十字/冷战十字/CSGO/圆形/菱形/三角形）+ RGB + 扩散 + 描边 + 旋转 |
| 💥 命中标识 | 击中/击杀特效 + 音效 + 边框 + 粒子 + 随机旋转（极性不重复）+ 击杀检测 |

## 特点

- **全部可自定义**：9 大模块，35 个配置子面板，每项独立开关
- **纯客户端**：零额外发包，不触发任何反作弊
- **中英文双语**：配置文件与 GUI 完整本地化
- **轻量化**：无捆绑、无依赖、无多余功能

## 配置

游戏内按 `O`（默认）打开配置 GUI，或通过 `mods` 列表进入。

> 适用于 PvP 方向。

## 技术栈

- Minecraft 1.8.9 / Forge 11.15.1.2318
- Java 8
- Gradle 4.10.3
